# Product Overview

Kortix is an open-source platform for building, managing, and training autonomous AI agents. The platform enables users to create specialized agents for various use cases, from general-purpose assistants to domain-specific automation tools.

## Core Components

- **Suna**: Flagship generalist AI worker that demonstrates platform capabilities through research, data analysis, browser automation, file management, and complex workflows
- **Agent Builder**: Visual tools for configuring, customizing, and deploying custom agents
- **Multi-platform**: Web dashboard (Next.js), mobile app (React Native/Expo), and Python SDK

## Key Capabilities

- Browser automation and web intelligence
- File and document management (docs, spreadsheets, presentations, code)
- System operations and command-line execution
- API integrations and cross-platform workflows
- Agent orchestration with LLM integration (Anthropic, OpenAI via LiteLLM)
- MCP (Model Context Protocol) tool integration
- Knowledge base and RAG capabilities
- Trigger-based automation (via Composio)

## Target Use Cases

- Customer service agents
- Content creation and marketing
- Research and data analysis
- Sales and CRM automation
- Industry-specific workflows (healthcare, finance, legal, education)
